# Toto Number Generator

## Overview

A simple web application to generate random numbers for the Toto lottery game. The application will display 6 unique numbers between 1 and 49 when the user clicks a button.

## Design and Features

### Style and Design
*   **Layout:** A centered, clean layout that is responsive and works on mobile and web.
*   **Color Palette:** A vibrant and energetic color palette.
*   **Typography:** Expressive and clear fonts to improve readability.
*   **Visual Effects:** Subtle shadows and gradients to create a sense of depth.
*   **Iconography:** Use of icons to enhance user understanding.
*   **Interactivity:** Interactive elements will have a "glow" effect on hover/focus.
*   **Background:** A subtle noise texture for a premium feel.

### Features
*   **Number Generation:** Generates 6 unique random numbers from 1 to 49.
*   **Display:** The generated numbers are displayed in a clear and visually appealing way.
*   **User Interaction:** A button to trigger the number generation.
